1. 單字沒熟透
2. 英語法習慣與中文不同 e.g. 
   1. 長安街 18 號 => no. 18, chan ann street. 
   2. 三分之一 => one third, 三分之二 => two thirds
   3. 13.87 => thirteen point eight seven
   4. 7:24 => seven twenty fore, 7:45 => a quarter to eight
   5. 姓名 (常見的要背), 難的可能會一字一字念
