## 音の法則のルール

これを英語だと言うのは少し偏りがあるかもしれません。ラテン語、ドイツ語、ギリシャ語などを含むインド・ヨーロッパ語族（印欧語）であり、すべて関連していると言うべきでしょう。拡張された単語は基本的に同じソースですが、地域や時間、その他の関係により音が異なります。

### 母音の法則

- `a, e, i, o, u の5つの母音の位置は交換可能` (例えば、sing, sang, snog, sung)
- `a, e, i, o, u + y, w 半母音の位置は交換可能`

### グリムの法則 & ヴェルネルの法則

> グリムの法則 (Grimm's Law)

| Rule |     |                 |            |           |
| ---- | --- | --------------- | ---------- | --------- |
| 1    | b   | p               | f / v / ph |
| 2    | d   | t               | th         | s (z) / c |
| 3    | g   | k (c / ch / qu) | h          |

> ヴェルネルの法則 (Verner's Law)

| Rule |                 |     |            |     |
| ---- | --------------- | --- | ---------- | --- |
| 1    | p               | b   | f / v      |
| 2    | t               | d   | th / s (z) |
| 3    | k (c / ch / qu) | g   | h          |

### その他の法則

- `l, r, n, m` は交換可能

- `s, z` は交換可能

- `u, v, w` は交換可能

## 例

> cap = head

- `cap`ital (Initial capital letter, capital city) 頭文字、首都
- `cap`tain (Captain, leader) キャプテン、リーダー
- `cab`bage (Vegetable `head`, cabbage) 野菜の`頭`、キャベツ
- `ch`ief (Leader, highest rank) リーダー、最高位
- `ch`ef (Chef) シェフ（料理長）
- `ch`apter (Chapter -> Main idea) 章 -> 主旨
- `c`e`p`halic (`Head`, skull) `頭`、頭蓋骨
- bi`c`e`p`s (Biceps -> Two `heads`, muscles) 上腕二頭筋 -> 2 つの`頭`、筋肉

> gen = birth

- `gen`e (Gene -> `birth`, heredity) 遺伝子 -> `誕生`、遺伝
- `gen`erate (`birth`, generate) `誕生`、生成する
- `gen`ius (Genius -> Born talent) 天才 -> 生まれつきの才能
- `gen`der (Gender -> Born gender) 性別 -> 生まれつきの性別
- `gen`erous (Generous -> Noble `birth`) 寛大な -> 高貴な`生まれ`
- `gen`uine (Genuine -> Natural `birth`, sincerity) 本物の -> 自然な`生まれ`、誠実
- `gn`aland (Innate, native) 先天的な、ネイティブな
- pre`gn`ant (Pregnant -> Before `birth`) 妊娠している -> `誕生`前
- `n`ative (Native -> `birth`) ネイティブ -> `誕生`
- `n`ature (Nature -> `birth`) 自然 -> `誕生`
- `n`aive (Naive -> Just `born`) 世間知らず -> `生まれた`ばかり
- `n`ation (Nation -> Place of `birth`) 国家 -> `誕生`の場所
- `k`in (Relatives -> Born together) 親戚 -> 一緒に生まれた
- `k`ind (Type, kind -> Born same kind) 種類、親切な -> 同じ種類として生まれた
- `k`ing (King -> Man of noble `birth`) 王 -> 高貴な`生まれ`の男

> p.l = fold

- sim`pl`e (`Fold` once -> Simple) 一回`折る` -> 単純な
- com`pl`ex (Folded together -> Complex) 一緒に折る -> 複雑な
- multi`pl`e (Many folds -> Multiple) たくさん折る -> 複数の、倍数
- du`pl`icate (Two folds -> Duplicate) 二回折る -> 複製する
- tri`pl`e (Three folds -> Triple) 三回折る -> 三倍の
- diplo`m`a (Official document folded in two -> Diploma) 二つ折りの公文書 -> 卒業証書
- em`pl`oy (Fold in -> Use) 中に折る -> 雇用する、使う
- de`pl`oy (Unfold -> Deploy) 展開する -> 配備する
- dis`pl`ay (Unfold -> Display) 展開する -> 表示する
- ex`pl`oit (Unfold -> Exploit) 展開する -> 開発する、搾取する
- im`pl`y (Fold in -> Imply) 中に折る -> 暗示する
- re`pl`y (Fold back -> Reply) 折り返す -> 返信する
- ap`pl`y (Fold to -> Apply) ～に折る -> 適用する
- com`pl`y (Fold together -> Comply) 一緒に折る -> 従う
- su`ppl`y (Fold under -> Supply) 下に折る -> 供給する

> spec = look

- `spec`troscope (Instrument to `look` at spectrum) スペクトルを`見る`ための器具 -> 分光器
- `spec`tacular (Spectacular -> Worth `looking` at) 壮観な -> `見る`価値がある
- `spec`tator (Audience -> Person `looking`) 観客 -> `見ている`人
- `spec`ulate (Speculate -> Look far -> Guess) 推測する -> 遠くを見る -> 推測
- `spec`trum (Spectrum -> Image seen) スペクトル -> 見える画像
- `spec`ies (Species -> Appearance seen) 種 -> 見える外見
- `spec`ial (Special -> Species usually seen is special) 特別な -> 通常見られる種は特別
- `sp`y (Spy -> Person looking) スパイ -> 見ている人
- `s`k`ep`tic (Skeptic -> Look carefully) 懐疑的な -> 注意深く見る
- `sc`ope (Range -> Range seen) 範囲 -> 見える範囲
- bi`sh`op (Bishop -> Person looking over -> Supervisor) 司教 -> 見渡す人 -> 監督者

> sta = stand

- `sta`nd (`Stand`) `立つ`
- `sta`tion (Station -> Place to `stand`) 駅 -> `立つ`場所
- `sta`ture (Stature -> Height `standing`) 身長 -> `立っている`高さ
- `sta`tus (Status -> Position `standing`) 地位 -> `立っている`位置
- `sta`ue (Statue -> Image `standing`) 像 -> `立っている`イメージ
- `sta`ble (Stable -> `Standing` steadily) 安定した -> しっかりと`立っている`
- `sta`blish (Establish -> Make `stand`) 設立する -> `立た`せる
- `st`ool (Stool -> Thing `standing`) スツール -> `立っている`もの
- `st`eeds (Steed -> Horse `standing` in stable) 軍馬 -> 厩舎に`立っている`馬
- `st`ud (Stud -> Horse `standing` in stable for breeding) 種馬 -> 繁殖のために厩舎に`立っている`馬
- sy`st`em (System -> `Standing` together) システム -> 一緒に`立っている`
- e`x`is`t` (Exist -> `Standing` out) 存在する -> 外に`立っている`
- in`s`is`t` (Insist -> `Standing` on) 主張する -> ～の上に`立っている`
- per`s`is`t` (Persist -> `Standing` through) 固執する -> ～を通して`立っている`
- re`s`is`t` (Resist -> `Standing` back) 抵抗する -> 後ろに`立っている`
- as`s`is`t` (Assist -> `Standing` by) 支援する -> そばに`立っている`
- con`s`is`t` (Consist -> `Standing` together) 構成される -> 一緒に`立っている`
- de`s`i`t` (Desist -> `Standing` down -> Stop) やめる -> 降りて`立つ` -> 停止
