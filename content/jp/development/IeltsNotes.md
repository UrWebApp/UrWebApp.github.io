## [IELTS 準備順序](https://www.douyin.com/video/7262565165500075283)

1. IELTS 公式サイトで関連情報を確認：申し込み手順、受験者への通知、申し込み時期、採点基準、試験内容
2. 自分の英語力を評価し、IELTS の過去問を解く時間を計る
   - ケンブリッジ IELTS 真題 11-15 から 1 セット選んで簡単なテストを行い、リーディング 3 つだけを行う
   - 合計 40 問中 32 問以上正解なら基礎は非常に強い、32-26 は普通、25 以下は悪い
   - 中国の大学英語試験 4 級・6 級（CET-4/6）との点数換算？
3. 準備期間を決定する（2-3 ヶ月）
   - 4 級 425 点以下：6 ヶ月準備、3 ヶ月基礎固め、3 ヶ月試験対策
   - 4 級 425-550 点：4 ヶ月準備、1 ヶ月基礎固め、3 ヶ月試験対策
   - 4 級 550 点以上：3 ヶ月試験対策
   - 6 級 425 点以下：3 ヶ月試験対策、1 ヶ月基礎固め、2 ヶ月試験対策
   - 6 級 425-550 点：2 ヶ月正式試験対策
   - 6 級 550 点以上：1 ヶ月正式試験対策
4. 教材選択
   - リーディング/リスニング：IELTS 語彙真経 + IELTS 語彙語根 + 連想記憶法
   - ライティング：顧家北が手取り足取り教える IELTS ライティング + 劉洪波ライティング真経 + Simon 満分模範解答（毎日 3 つのトピックを暗記、3 週間後には良い効果）
   - リスニング：ケンブリッジ IELTS 真題 4-18 + 王陸語料庫。質問タイプ別に練習し、その後セット全体を解く
   - スピーキング：IELTS スピーキング問題バンク。1 月、5 月、9 月に更新される。バンク内の問題への回答を準備する
   - リーディング：劉洪波リーディング真経 + オンライン講座 + 同義語置換

## 文法 [英語兎](https://www.youtube.com/watch?v=is7vn5URVcc&ab_channel=%E8%8B%B1%E8%AF%AD%E5%85%94)

- 台湾では、目的語（Object）を「受詞」と解釈します

```
                  => 述語動詞 => 単文（文の構成要素 主語/動詞/目的語/補語/連体修飾語/連用修飾語/同格...）組み合わせ 重文/複文
英語の核心 => 動詞      => 複文（主節/従属節） 従属節（動詞、名詞、代名詞、冠詞、形容詞、数詞、副詞、前置詞、感嘆詞、接続詞）
                     => 時間/状態/法/否定/受動/可能性/義務を表すが、助動詞が必要

                  => 準動詞（非述語動詞） => 変更された述語動詞、ほぼすべての従属節を置き換え、文を簡素化できる
```

> 主語（Subject / 何 / 人または物）+ 述語（Predicate / どうした / 広義の動作、何が起こったか）

- 核となる概念：述語内の動詞は、意味を完全に表現するためにいくつの対象を必要とするか

1. 主語 + （自動詞）動詞 || 受け手は不要

   広義の動作 / 動詞：

   `単独で完了する` / 自動詞 => 父ウサギ（寝る）

2. 主語 + （他動詞）動詞 + 目的語 || 受け手が 1 つ

   `動作の受け手が "1つ" ある` => 父ウサギは（好き）？ => 父ウサギは（あなたを）好き 目的語 Object

3. 主語 + （二重他動詞）動詞 + 直接目的語 + 間接目的語 || 受け手が 2 つ

   `動作の受け手が "2つ" ある` => 父ウサギは（あなたに）（英語を）教える

   父ウサギは（英語を 直接目的語）教える

   父ウサギは（あなたに 間接目的語）教える

   父ウサギは（あなたに）（英語を）教える

4. 主語 + （複合他動詞）動詞 + 目的語 + 目的語補語 || 受け手が 1 つ

   `動作の受け手が "1つ" あるが、この動作だけでは意味のある文を構成できない（複合他動詞）`

   父ウサギはあなたを"みなす"？

   父ウサギはあなたを（賢いと / 目的語補語 / Object Complement）みなす

5. 主語 + （連結）動詞 + 主語補語/表語 || 情報を付与する

   連結動詞（Linking Verb）

   is => 前述の単語に後続の単語の情報を付与する

   父ウサギは背が"高い" => 父ウサギ = 高い

   父ウサギは部屋の"中にいる" => 父ウサギ = 部屋の中

   父ウサギは背が高く"見える（looks）" => 父ウサギ = 背が高い

### 文の構成要素 Parts Of Speech

単文（Simple Sentence） => これ以上文を分割できない

1. 主語 Subject
2. 述語動詞 Predicate Verb
3. 目的語 Object
4. 目的語補語 Object Complement
5. 主語補語 Subject Complement / 表語 Predicative
6. 連体修飾語 Attributive：主語または目的語を修飾する => （小さい AT）ウサギ S は（大きな AT）ニンジン O を食べた PV
7. 連用修飾語 Adverbial：述語動詞を修飾する => ウサギ S は（素早く AD）食べた PV
8. 同格 Appositive：主語または目的語を再度述べる => 父ウサギ S、（英語教師 AP、主語と同等の地位）、はニンジン O を食べた PV

単文の組み合わせ => 重文（Compound Sentence） / 入れ子の複文（Complex Sentence）

> 複文（Complex Sentence） => 主節（main clause） + 従属節（subordinate clause）
> 従属節（subordinate） は、単文を別の文の主語や目的語に置き換えるものです。名詞節（主語節/目的語節/主語補語節/同格節）、形容詞節、副詞節

### 品詞

文の構成方法は同じ（主語・述語・目的語）だが、構成する品詞が異なる

1. 名詞 Nouns：人や物を表す => 固有名詞 / 普通名詞 ウサギ
2. 冠詞 Articles：人や物を説明する => 父ウサギは"一匹の a"ウサギである
3. 代名詞 Pronouns：人や物の代わりになる => "私 I"はウサギである
4. 形容詞 Adjectives：人や物を形容する => 私は"賢い"ウサギである
5. 数詞 Numerals：数量を表す => 私はニンジンを"2 本"食べた
6. 副詞 Adverbs：動作や形容詞を修飾する => 私は"素早く"ニンジンを食べた
7. 前置詞 Prepositions：他の語との関係を表す => 私は箸"で with"ニンジンを食べた
8. 感嘆詞 Interjections：感嘆を表す => "ああ"、このニンジンは本当に美味しい
9. 接続詞 Conjunctions：単語と文をつなぐ => お腹が空いていた"ので because"、私はニンジン"と and"ジャガイモを食べた
10. 動詞 Verbs

---

文の構成要素と品詞は別物です

### 述語動詞

時制・相・法（Tense / Aspect / Mood）

- 時制

1. 動作の"時"間を表す => 現在/過去/未来/過去のある時点から見た"将来"（過去未来）
2. 動作の状"態"（相）を表す => 明示なし（一般）/完了/進行/完了かつ継続（完了進行）

4\*4 => 16 種類の時制

3. 動詞の法 => 動作の仮定/感情を表す
   - 仮定法：願望/事実と異なることを表す => もし私がウサギ"だったら" => 実はウサギではないことを暗示
   - 直説法：私はニンジンを"食べる"
   - 命令法：このニンジンを"食べろ"

> 述語動詞は単独で否定/可能性/必須などを表すことはできません...

### 助動詞 Auxiliary Verbs

> 偽装された実義動詞、あるいは他の品詞 =>
> can できる/缶、might かもしれない/力、must カビが生える/しなければならない、do する、have 持つ、be 存在

食べる Eat =>

食べた have eaten

食べている is eating

食べられた is eaten

食べることができる can eat

食べるかもしれない might eat

食べなければならない must eat

食べない do not eat

### 準動詞（非述語動詞）

動詞は述語動詞になることができますが、主語、主語補語、連体修飾語、目的語、連用修飾語、目的語補語、同格にもなれます

準動詞はすべての従属節を置き換えることができ、それによって文を簡素化できますが、動作の時間、状態、態、法を表すことはできません

- 不定詞：to eat（原形と同じ）
- 現在分詞：eating（進行形と同じ）
- 動名詞：eating（進行形と同じ）
- 過去分詞：eaten（完了形と同じ）
